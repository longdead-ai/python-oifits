python-oifits
=============

Python Module to Read/Write OIFITS 

Version : 0.1.0

How to install:
--------------

  $ pip install oifits
  

Why python-oifits?
------------------

This module has been created as a part of my major technical project. I was not able to find any module to work with OIFITS data easily. Hence, created `python-oifits`.
